## Buddy Backend

Back-end module of the [Buddy System](http://146.56.190.179/)

[![Go Reference](https://pkg.go.dev/badge/github.com/kmu-kcc/buddy-backend.svg)](https://pkg.go.dev/github.com/kmu-kcc/buddy-backend)

### Install dependencies

  ```bash
  go mod tidy
  ```

### Run

  ```bash
  ./launch.sh
  ```
